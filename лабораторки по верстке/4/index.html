The Path to Enlightenment

These instructions are for *nix platforms. We also have Windows intstructions.

You can run the test by calling the path to enlightenment.rb file.

In your terminal, while in the ruby koans directory, type:

[ ruby_koans ] $ ruby path_to_enlightenment.rb

Red, Green, Refactor

In test-driven development(TDD) the mantra has always been red: write a
failing test and run it, green: make the test pass, and refector:look at the code
and see if you can make it any better.
